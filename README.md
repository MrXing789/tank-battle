# 坦克大战 (Tank Battle)

一个使用纯 HTML、CSS 和 JavaScript 制作的坦克大战网页游戏。

## 在线试玩

直接打开 `index.html` 即可开始游戏。

## 操作说明

- **W / A / S / D** 或 **方向键**：移动坦克
- **鼠标**：瞄准
- **鼠标左键** 或 **空格键**：发射炮弹

## 游戏规则

- 消灭敌方坦克获得分数
- 敌人会随着波次增强，刷新速度也会加快
- 生命值为 0 时游戏结束

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- CSS3

## 本地运行

```bash
git clone <repository-url>
cd tank-battle
# 用浏览器打开 index.html
```
